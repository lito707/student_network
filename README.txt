The project is installed under a a Linux OS Ubuntu 14.04 64-bit

To set up the project the following are needed to be installed:

1) Install Python 2.7
-- A guide to install python can be found in this link
https://www.python.org/downloads/release/python-2710/

2) Install Django -- Django 1.8.4 was used during development--
	- To install it with pip 
		pip install Django==1.8.5
-- Or follow the guide provided to install Django Project 
https://www.djangoproject.com/download/


3) Install Django Activity Stream
	- To install via pip
		pip install django-activity-stream
-- Or follow the guide in the link provided
http://django-activity-stream.readthedocs.org/en/latest/installation.html
